Map

Parent Userdata
Library native.*
Revision Current Release (2020.3601)
Keywords map, map view
See also native.newMapView()

Overview

The map view feature lets you integrate one or more map views into your app. It supports various methods of two-way communication between the map contents and the surrounding application, including address or landmark lookup, marking the map with pins, and reverse geocoding (converting latitude/longitude into the closest street address).
