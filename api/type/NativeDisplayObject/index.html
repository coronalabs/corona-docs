NativeDisplayObject

Parent DisplayObject
Library native.*
Revision Release 2024.3703
Keywords native object, native UI

Overview

Native display objects are separate from Corona’s OpenGL drawing canvas and always appear in front of normal display objects.

See the native.* library for a list of all native APIs.
