## Overview

This is a translated response from the licensing server. The value may be one of the following strings:

* `"Licensed"`
* `"Not licensed"`
* `"Invalid public key"`
* `"Missing permission"`
* `"Invalid package name"`
* `"Non matching UID"`
* `"Not market managed"`
* `"Error contacting server"`
