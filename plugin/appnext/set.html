appnext.set*

Type Functions
Revision Release 2024.3703
Keywords ads, advertising, Appnext, set
See also appnext.get*
appnext.*

Overview

When you create an Appnext ad via appnext.createAd(), you can include a wide array of parameters to customize the look and behavior of the ad.

Following creation of the ad, all parameters can also be set by calling the following functions.

Notes
  • Certain parameters apply only to specific ad types, per the indicated sections.
  • Parameters can also be read/retrieved via corresponding appnext.get* functions.

General

  • appnext.setCategories()
  • appnext.setPostback()
  • appnext.setButtonText()
  • appnext.setButtonColor()
  • appnext.setPreferredOrientation()
  • appnext.setBackButtonCanClose()
  • appnext.setMute()
