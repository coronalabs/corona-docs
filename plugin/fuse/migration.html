AdRally to Fuse Migration Guide

Overview

If you previously used the AdRally™ plugin, this guide will help you migrate to the new Fuse plugin.

Project Settings

Please refer to the Project Settings section in the Fuse documentation.

Function Changes

The Fuse plugin completely replaces the AdRally™ plugin. In addition to the namespace change, there are some minor changes to several functions:

  • fuse.init() — The providerName argument has been removed. Also, if you build with Corona SDK, you only need to supply the listener function. The API key from the Fuse dashboard does not need to be specified, as the server will assign an API key based on your app’s bundle/package ID.

  • fuse.checkLoaded() — The adType argument has been removed. If provided, a warning message will appear in the console.

  • fuse.load() — The adType argument has been removed. If provided, a warning message will appear in the console.

  • fuse.show() — The adType argument has been removed. If provided, a warning message will appear in the console.
