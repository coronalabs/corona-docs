Apple Game Center

Type Library
Revision Current Public Release (2018.3326)
Keywords gameNetwork, Game Center
Platforms iOS
Sample https://github.com/coronalabs/plugins-sample-gamenetwork-apple

Overview

Game Center lets friends in on the action with leaderboards and achievements. The nomenclature used in the Corona APIs for Game Center attempt to match the official Game Center APIs as much as possible, allowing you to cross-reference with official Game Center documentation.

Gotchas

Game Center is not supported in the Corona Simulator.

Syntax

local gameNetwork = require( "gameNetwork" )

Functions

gameNetwork.init()

gameNetwork.request()

gameNetwork.show()

Project Settings

To use this plugin, add an entry into the plugins table of build.settings. When added, the build server will integrate the plugin during the build phase.

settings =
{
    plugins =
    {
        ["CoronaProvider.gameNetwork.apple"] =
        {
            publisherId = "com.coronalabs"
        },
    },
}
